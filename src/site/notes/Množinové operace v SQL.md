# Množinové operace v SQL
#card 
**UNION** - sjednocení
**INTERSECT** - průnik
**EXCEPT** - množinové mínus
**UNION ALL** - jak UNION, ale neřeší duplicity (nevolá [[DISTINCT\|DISTINCT]]){ #1777372339666}
